<h3 style="text-align:center;">Криптосистема с открытым ключом</h3>
<p>Для его построения каждый из абонентов – предположим, что это абонент A, – выбирает два простых числа p<sub>1</sub> и p<sub>2</sub> (на практике – очень большие) и находит их произведение r<sub>A</sub> = p<sub>1</sub> ∙ p<sub>2</sub>.</p>
<p>Вычислив φ(r<sub>A</sub>) = (p<sub>1</sub> − 1) ∙ (p<sub>2</sub> − 1), абонент A выбирает свой открытый ключ a из условий a ∈ ℕ, (a, φ(r<sub>A</sub>)) = 1, 0 < a < φ(r<sub>A</sub>).</p>
<p>Секретный ключ α определяется теперь однозначно из условий α ∈ ℕ, a ∙ α ≡ 1 (mod φ(r<sub>A</sub>)), 0 < α < φ(r<sub>A</sub>).</p>
<p>Аналогичные действия выполняет любой другой абонент, скажем, абонент B.</p>
<p>Таким образом, на предварительном этапе работы абоненты A и B обладают следующей информацией:</p>
<p>A: p<sub>1</sub>, p<sub>2</sub> ∈ P большие; r<sub>A</sub> = p<sub>1</sub> ∙ p<sub>2</sub>; φ(r<sub>A</sub>) = (p<sub>1</sub> − 1) ∙ (p<sub>2</sub> − 1),</p>
<p>Открытый ключ: a ∈ ℕ, (a, φ(r<sub>A</sub>)) = 1, 0 < a < φ(r<sub>A</sub>).</p>
<p>Секретный ключ: α ∈ ℕ, a ∙ α ≡ 1 (mod φ(r<sub>A</sub>)), 0 < α < φ(r<sub>A</sub>).</p>
<p>B: q<sub>1</sub>, q<sub>2</sub> ∈ P большие; r<sub>B</sub> = q<sub>1</sub> ∙ q<sub>2</sub>; φ(r<sub>B</sub>) = (q<sub>1</sub> − 1) ∙ (q<sub>2</sub> − 1),</p>
<p>Открытый ключ: b ∈ ℕ, (b, φ(r<sub>B</sub>)) = 1, 0 < b < φ(r<sub>B</sub>).</p>
<p>Секретный ключ: β ∈ ℕ, b ∙ β ≡ 1 (mod φ(r<sub>B</sub>)), 0 < β < φ(r<sub>B</sub>).</p>
<p>В открытый доступ абонент A (как и любой другой абонент системы) предоставляет используемый им модуль r<sub>A</sub> = p<sub>1</sub> ∙ p<sub>2</sub> и открытый ключ a.</p>
<p>Сами простые числа p<sub>1</sub> и p<sub>2</sub> (а, следовательно, и φ(r<sub>A</sub>) = (p<sub>1</sub> − 1) ∙ (p<sub>2</sub> − 1)) и ключ α абонент A держит в секрете.</p>
<ol>
<li>
<p>Абонент A составляет сообщение для абонента B и кодирует его в виде числа m, 0 < m < r<sub>B</sub>. Затем абонент A шифрует сообщение m открытым ключом b абонента B:</p>
<p>m<sub>1</sub> ≡ m<sup>b</sup> (mod r<sub>B</sub>), 0 < m<sub>1</sub> < r<sub>B</sub>;</p>
<p>Полученное сообщение m<sub>1</sub> абонент A отправляет абоненту B.</p>
</li>
<li>
<p>Абонент B расшифровывает сообщение m<sub>1</sub> своим секретным ключом β:</p>
<p>m<sub>2</sub> ≡ m<sub>1</sub><sup>β</sup> (mod r<sub>B</sub>), 0 < m<sub>2</sub> < r<sub>B</sub>.</p>
<p>В этом случае m<sub>2</sub> = m, и абонент B получает возможность прочитать исходное сообщение от абонента A, переведя число m в буквенную форму.</p>
</li>
</ol>
